# 2025 CMEE Bootcamp Coursework Repository

## Description

This repository contains coursework completed during the 2025 CMEE Bootcamp at Imperial College London. The repository is organized by coursework completed each week.

## Repository Structure

The coursework is organized into weekly folders:

- **Week 1**: Bash scripting 
- **Week 2**: Python programming
- **Week 3**: R programming

Each week's folder contains:
- `/code`: Scripts and programs developed during that week
- `/data`: Input data files used for exercises (if applicable)
- `/results`: Output files and analysis results, note that these are not uploaded and you will have to generate your own results using the code provided.

## Languages

This project uses the following languages:

- **Bash** (Unix shell scripting) - Version 4.0+
- **Python** - Version 3.8+
- **R** - Version 4.0+
